# Project Specular Website

A narrative-driven website for Project Specular by EvenEi - Everyday Intelligence designed to live with you.

## Overview

This website is built with Next.js 16 and showcases Project Specular, a revolutionary smart eyewear product that builds a private, contextual memory of your life. The website follows a carefully crafted narrative structure designed to engage users through a mysterious, intelligent, and confident experience.

## Design Philosophy

- **Theme**: Deep black background with subtle grain texture and glowing emerald green accents
- **Mood**: Quiet, intelligent, mysterious, confident
- **Goal**: Make users feel observed, curious, and understood
- **Rule**: Never explain everything at once

## Features

### Narrative Sections

1. **Hero Section** - Opening animation with timed text reveals
2. **The Hook** - Introducing the problem of lost memories
3. **What This Is** - Defining Project Specular without over-explaining
4. **Everyday Intelligence** - Explaining the core concept
5. **The Problem** - Why current technology feels wrong
6. **The Shift** - From assistance to awareness
7. **Memory** - The core feature
8. **Asking Questions** - Natural interaction examples
9. **Live Companion Mode** - Independent awareness
10. **Control & Trust** - Privacy and user control
11. **How It Works** - Technical overview
12. **The Products** - Three tiers (Basic, Pro, Max)
13. **Why This Exists** - The mission
14. **Final Section** - Call to action

### Technical Features

- Scroll-based narrative experience
- Smooth animations and transitions
- Responsive design (mobile and desktop)
- Hero section with timed fade-in effects
- Grain texture overlay for atmospheric effect
- Emerald green accent color throughout
- Product pricing cards with hover effects

## Tech Stack

- **Framework**: Next.js 16.0.7
- **React**: 19.2.1
- **Styling**: Tailwind CSS v4
- **TypeScript**: Type-safe development
- **Fonts**: Geist Sans and Geist Mono from Google Fonts

## Getting Started

### Prerequisites

- Node.js 20+ installed
- npm or yarn package manager

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The application will be available at `http://localhost:3000`
